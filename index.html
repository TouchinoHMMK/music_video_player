🎵 MediaBox

URLからダウンロード

YouTube / ニコニコ動画などのURLを貼ると、GitHub Actionsがダウンロードしてリポジトリに保存します(完了まで1〜3分)。
プレイリストのURLを貼ると中の曲をすべて取得します。同じ名前の曲・取得済みの曲は自動でスキップされます。

実行状況

ファイルをアップロード

手元のmp3/mp4ファイルを直接リポジトリに追加します(1ファイル最大約50MB)。

かんたん設定(2台目以降)

設定済みの端末で「設定コードをコピー」→ 新しい端末でここに貼り付けるだけで完了します。

GitHub連携設定

リポジトリ情報は入力済みです。初回はトークンだけ入力してください。

詳細設定(通常は変更不要)

トークンは Fine-grained PAT で、このリポジトリに対して Contents: Read and write / Actions: Read and write の権限を付けてください。トークンはこの端末のブラウザ内(localStorage)にのみ保存されます。
